## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/nithesh-shetty9/react-journey.git
```

Navigate to the project:

```bash
cd react-journey
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
